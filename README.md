# Volleyball Tournament Organizer

Currently only in german

## Idee
Optimal um Turniere zu organisieren, bei denen jede Runde zufällige Teams spielen.
Anpassbar sind die Teamgröße, die Anzahl der Felder, die Spieldauer und ein Fairness-Faktor, der bestimmt, wie groß die Stärke-Differenz zwischen zwei Teams maximal sein darf.

## Verwendung
Das Repository klonen oder downloaden (und entpacken). Danach die Datei ``project/launch.html`` im Browser öffnen.  
Sollte der Browser-Tab geschlossen werden, geht jeder Fortschritt verloren.  
Alle Einstellungen gelten ab dem Moment der Änderung.
